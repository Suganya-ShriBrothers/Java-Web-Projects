
# Retailer Reward Program for Customer

This Project is mainly for Retailers. Retailer will offer reward points to each customer based on the purchase for every month. 


## Requirements
As per requirement each customer will receive 2 points for each dollar spent over $100 in each transaction and 1 point for each dollar spent between $50 and $100 in each Transaction.
We have to maintain record for each transaction for each customer during 3 month of period. Calculate reward points based on per month transaction and make a total.


## Recommended Technology
Java 8, Spring Boot, Microservice and MySQL

## Installation
Need to install Jdk8, eclipse,mySql,git
